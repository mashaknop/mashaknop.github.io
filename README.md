# Maria Knop - Portfolio

This is the source code for Maria Knop's portfolio website. The site is built using [Jekyll](https://jekyllrb.com/).

## How to render HTML locally

### Prerequisites

Ensure you have Ruby installed on your system.

1. Install Bundler if you haven't already:
   ```bash
   gem install bundler
   ```

2. Install the required dependencies:
   ```bash
   bundle install
   ```

### Running the server

To run the site locally and watch for changes, use the following command:

```bash
bundle exec jekyll serve
```

The site will be available at [http://localhost:4000](http://localhost:4000).
